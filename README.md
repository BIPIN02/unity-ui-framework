
# Unity UI Framework

Reusable Unity UI architecture system.

## Features
- Screen navigation
- Popup manager
- Loading screen
- Toast notification
- Reusable UI system

## Demo Setup

Create:
- Canvas
- HomeScreen Panel
- ShopScreen Panel
- SettingsPopup Panel
- Toast Panel

Attach:
- UIManager.cs
- PopupManager.cs
- DemoController.cs

## Buttons
- Open Shop
- Open Home
- Open Settings
- Show Toast
